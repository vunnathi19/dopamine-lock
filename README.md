🔒 Dopamine Lock — Productivity Web App

Dopamine Lock is a productivity-focused web application designed to reduce procrastination and improve self-discipline through a reward-based task system. Users must complete tasks or focus sessions to unlock entertainment, encouraging structured work habits and better time management.

🚀 Features
📋 Task management system
⏳ Focus session-based unlocking mechanism
🔒 Entertainment lock until tasks are completed
🎯 Reward-driven productivity flow
⚡ Simple and distraction-free UI
📱 Responsive frontend design
🧠 Core Idea

The app is built on the principle of delayed gratification:

Entertainment is earned, not freely accessed.

Users stay focused by completing assigned tasks or focus sessions before accessing distractions.

🛠 Tech Stack
HTML5
CSS3
JavaScript (Vanilla)
🎯 Purpose

This project was built as a frontend learning project to explore:

DOM manipulation
UI/UX design principles
State-based interaction systems
Productivity psychology concepts
